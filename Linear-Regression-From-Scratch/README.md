# Linear Regression From Scratch

This project implements Linear Regression using only NumPy and Gradient Descent.

## Features

- Linear Regression from Scratch
- Gradient Descent
- Mean Squared Error (MSE)
- Weight and Bias Optimization
- Student Marks Prediction
- Data Visualization using Matplotlib

## Dataset

Input:
- Hours Studied

Output:
- Marks Obtained

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Run

```bash
pip install -r requirements.txt
python linear_regression.py
```
