📊 Student Marks Predictor using Linear Regression
📌 Project Overview

This project implements a simple Machine Learning model from scratch using NumPy to predict student marks based on the number of hours studied.

The model uses Linear Regression with Gradient Descent optimization to learn the best-fit line between study hours and marks.

🧠 Objective:
To understand how machine learning works internally by manually implementing:
Loss function (Mean Squared Error)
Gradient Descent
Weight (w) and bias (b) optimization
Prediction using learned parameters

⚙️ Technologies Used
Python 🐍
NumPy (Numerical computations)
Pandas (Data handling)
Matplotlib (Visualization)
